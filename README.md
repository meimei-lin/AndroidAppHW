### 作業2:LinearLayout練習-排數字1~5

### 作業4:隨機顏色與合成

* 需顯示切換路徑

* 輸入美金和人民幣兩種外幣匯率，使用SQLite資料庫存外幣兌換匯率，從資料庫取得匯率，在勾選兌換成美金或人民幣後，輸入外幣金額，即可換算和顯示兌換的新台幣金額
